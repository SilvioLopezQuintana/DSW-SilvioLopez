Propósito del sitio

El propósito del sitio es fortalecer las estrategias educativas relacionadas con el diseño de sitios web,con una introducción en el uso de etiquetas básicas de HTML,atributos y valores.

Etiquetas básicas de HTML

La etiqueta es la unidad básica en HTML, se identifica por estar rodeada por corchetes angulares(< >), donde el inicio de la etiqueta se indica (< xxx) y el cierre (< / xxx >). Dentro de las etiquetas básicas tenemos : header, footer, article, nav,section.

Romero Alamán, J. (2017). M-flashcarding como recurso de aprendizaje del HTML.

Atributos y valores

Algunas etiquetas necesitan de parametros (atributos) para funcionar correctamente,estos atributos se introducen dentro de la etiqueta, por ejemplo si se desea insertar una imagen con cierto tamaño se utiliza width = 300px (donde width es el atributo que se refiere al tamaño y 300px es valor de dicho tamaño en pixeles).

Romero Alamán, J. (2017). M-flashcarding como recurso de aprendizaje del HTML.
