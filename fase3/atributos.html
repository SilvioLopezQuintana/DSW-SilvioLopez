Atributos y valores

Los atributos en HTML van siempre dentro de la etiqueta de apertura y se utilizan para
agregar alguna característica al contenido.

A todo atributo se le debe especificar un valor, el siguiente es un ejemplo de una etiqueta
con atributo y su respectivo valor:
< p width = "400px" height = "200px" > Este es un párrafo < / p >.

Para este caso la etiqueta es un párrafo al cual se le define el atributo ancho (width) de 400
pixeles, y una altura (height) de 200pixeles, y el párrafo tiene la frase: Este es un párrafo.
