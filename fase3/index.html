Propósito del sitio web

El propósito es fortalecer las estrategias educativas relacionadas con el diseño de sitios web, dando una introducción en el uso de etiquetas básicas de HTML, atributos y valores.

Este sitio web se diseñó utlizando lenguaje HTML5 , y los estilos fueron desarrollados con CSS3
